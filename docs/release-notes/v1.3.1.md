## RayMol 1.3.1

A patch release fixing camera actions on the Metal renderer.

- **Fixed: Zoom / Orient / Center now animate.** The Objects-panel **A** (Actions) menu's camera commands run as animated view moves, but the Metal renderer wasn't advancing the animation each frame, so they appeared to do nothing. They now play to their target. (This fix existed for 1.3.0 but was left out of the shipped 1.3.0 build; 1.3.1 ships it.)

Includes everything from 1.3.0: outline color & width controls, one-click AI connect (Claude Code / Claude App) with Copy connection details, the background-swatch fix, and the offscreen/transparent-background export fixes.

Built on the open-source PyMOL engine. Updates install automatically via the in-app updater (**Check for Updates…** in the app menu).
