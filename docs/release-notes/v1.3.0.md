## RayMol 1.3.0

New rendering controls, one-click AI connection, and export fixes.

- **Outline color & width.** The real-time silhouette outline is now fully customizable — choose any contour color and set its thickness right from the Scene panel (it was previously fixed at a thin black line).
- **One-click AI connect.** The RayMol MCP menu adds **Claude Code** and **Claude App** quick-connect buttons, plus a **Copy connection details** button — no more walking through the setup sheet by hand.
- **Fixed: background color swatch.** After choosing a background with the color picker, the swatch could incorrectly show black; it now reflects the actual background color.
- **Fixed: image export dropping the surface.** Offscreen and transparent-background PNG exports no longer cut out the molecular surface (a race with the deferred surface build).
- **Cleaner update notes.** Release notes now render as formatted text directly in the in-app updater dialog.

Built on the open-source PyMOL engine. Updates install automatically via the in-app updater (**Check for Updates…** in the app menu).
